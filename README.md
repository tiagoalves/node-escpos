node-escpos
=================

This is a library used for communicatng with ESCPOS compatible devices.

Quick start
-----------

**For Printers**

printLine(text)  
printCentered(text)  
printCenteredLen(text, charOffSet)  
printCommand(escposCommand)  

**For Client Displays**

centeredUpperLine(text)  
centeredBottomLine(text)
